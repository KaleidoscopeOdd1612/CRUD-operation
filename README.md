# Express Crash Project

## Clone project
```bash
git clone https://github.com/KaleidoscopeOdd1612/CRUD-operation.git
```

## Run frontend and backend

### Backend
```bash
cd server
npm install
node server.js
```

### Frontend
```bash
cd client
npm install
npm run dev
```

### Open in browser
```
http://localhost:5173
```
